Aprovação em Unidades

Foi pedido a um programador que escrevesse um programa para ler a unidade em que um aluno está e a sua média de aprovação. O programa deve imprimir uma mensagem com a média de aprovação e a próxima unidade habilitada para o aluno. Esse programa deveria funcionar da seguinte maneira.

$ python joao.py
Unidade? 2
Média de aprovação na unidade? 8.23

O aluno vai para a unidade 3 com média 8.2.

Observação Atente para a linha em branco na saída após leitura da média de aprovação e antes da linha final da saída.

Infelizmente, o programa feito pelo programador não funciona como esperado, pois contém alguns erros. Pede-se que você identifique os problemas do programa fornecido e o modifique para que funcione como o esperado.

# coding: utf-8
unidade = raw_input('Unidade? ')
media = raw_input('Média de aprovação na unidade? ')

proxima_unidade = unidade + 1
msg = 'O aluno vai para a unidade %d com média %.2f.'

print msg % (proxima_unidade, media)
